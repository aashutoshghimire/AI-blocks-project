# To run this file:
-------------
```
1. Clone or download the .ipnb file in your computer.
2. Go to https://colab.research.google.com/ in your web browser
3. Go to File -> New Notebook -> Upload
4. Select the downloaded .ipynb file
5. Press shift and enter together to run each cells and see the output
```

## Alternative:
--------------
```
1. Clone or download the .ipnb file in your computer.
2. Install python and jupyter notebook in your computer
3. Open jupyter notebook in your computer
4. Go to File -> Upload
4. Select the downloaded .ipynb file
5. Press shift and enter together to run each cells and see the output
```
